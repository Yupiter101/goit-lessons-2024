Блокова модель

  1. box-sizing: content-box | border-box | inherit В інструментах розробника, на вкладці Computed
  2. modern-normalize (підключено тут)
  3. Геометрія: верх (top), право (right), низ (bottom) і ліво (left)
  4. padding: 10px 15px 20px 25px;
  5. margin-bottom: 20px
  6. border: 5px solid tomato;
  7. border-radius: 10px;
  8. Дізнатися тип будь-якого елемента можна в довіднику
  9. display: block
  10. Горизонтальне центрування блокових елементів
  11. margin-right: auto; margin-left: auto;
  12. or margin: 0 auto;
  13. Рядкові елементи display: inline Їх ширина і висота залежать тільки від вмісту, явно задати їх не можна. Тобто властивості width і height не мають жодного ефекту. Тобто вони ігнорують значення верхніх і нижніх margin, padding і border Зате left right працюють
  14. Картинкам надати примусово display: block;
  15. inline-block/ a href="" class="link"/ display: inline-block;/ можна задати вертикальні поля та іншу геометрію.
  16. Ховаємо елементи display: none / не можна анімувати / .is-active (js)
  17. Псевдокласи :first-child і :last-child / Псевдоклас :last-child вибирає останній елемент в колекції сусідів. / .list-item {margin-bottom: 12px;} / .list-item:last-child {margin-bottom: 0; }
  18. Аналогічно працює :first-child, застосовуючи стилі до першого елемента в колекції сусідів.
  19. .list-item:not(:last-child) {}
  20. Псевдоклас :nth-child(an+b)
  21. Наприклад, якщо a = 2, а b = 1, то an+b = 2n+1. Ця формула вибере всі непарні елементи.
  22. N-й елемент .list-item:nth-child(5) {}
  23. Парні і непарні елементи .list-item:nth-child(2n) {}
  24. Від N-го елемента .list-item:nth-child(n + 6) {}
  25. Кожен N-й елемент .list-item:nth-child(3n + 1) {}
