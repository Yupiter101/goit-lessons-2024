Базове відео html

Базові інструменти html для відео

https://video-converter.com webm

Приклад

<video
    poster="../img/fonts.JPG"
    width="600"
    controls
    >
    <source src="../../vid/VID_Illia.webm" type="video/webm"/>
    <source src="../../vid/fonts.mp4" type="video/mp4"/>
    <source src="../../vid/fonts.mp4" type="video/ogg"/>
</video>

Параметри відео:

  • poster="../../img/fonts.JPG"
  • width="600"
  • height="300" (Не обовязково)
  • muted - audio off
  • autoplay
  • controls - panel
  • loop
  • preload: none | metadata = ok | auto
