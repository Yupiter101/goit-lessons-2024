Import / Export

Import та Export працюють лише якщо вказати type="module" при підключенні js.файлу. Та в кінці вказано розширення ".js" 

import {my_W, my_W1} from '../js/admins.js';

module.exports require - Не працює

У версії з Parcel працює усе, але теж з type="module"

    <script src="../../js-exampl/import.js" type="module"></script>

Відкривай консоль F12

Імпорт у файл import.js з файлу export.js

Зміст

  1. V1 - export const | from './'
  2. V2 - export default
  3. V3 - module.exports require - не працює

Ніхрена не працює

export const

Це працює але не забути поставити type="module"

    export const my_A = 44;
    export const my_A1 = 45;

    import {my_A, my_A1} from './export.js'; 
    console.log(my_A);// Not work
    console.log(my_A1);// Not work

export default

Це працює але не забути поставити type="module"

        const my_B = 55;
        export default my_B;

        import asd from './export.js';
        console.log(asd);

module.exports

Це не працює. Навіть з type="module"
