CSS-змінні

  • :root { --brand-color: #3f51b5; }
  • .section-title { color: var(--brand-color); }

  • Тут tomato за змовчуванням якщо нема іншого

    .section-title { color: var(--brand-color, tomato); }
  • Область видимості
  • .alert { --alert-text-color: black; --alert-accent-color: tomato; }

Оформлення тексту

  • text-decoration: none | underline | line-through | overline
  • text-transform: none | uppercase | lowercase | capitalize
  • text-align: left | right | center | justify
  • text-indent: значення | відсотки | inherit (відступ абзацу)
  • line-height: множник | значення | відсотки | normal | inherit
  • letter-spacing: значення | normal | inherit
  • word-spacing: значення | normal | inherit
  • text-shadow: x, y, радіус, колір
  • white-space: normal | nowrap | pre | pre-wrap
